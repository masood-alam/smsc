Name
	smsc mproc remove

SYNOPSIS
	smsc mproc remove <id>

DESCRIPTION
	This command is used to show all the message processing rule. 

PARAMETERS

		Optional Parameters

	id				- A unique id for which message processing rule is to be shown. 
																		  
	
SEE ALSO
	smsc mproc add, smsc mproc modify, smsc mproc show
