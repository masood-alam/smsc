Name
	smsc get deliverypause

SYNOPSIS
	smsc get deliverypause

DESCRIPTION
	This command is used to get the deliverypause value. 
	If true, delivery of all SMS is paused by SMSC.

EXAMPLES
	smsc get deliverypause
