Name
	smsc get maxduedelay

SYNOPSIS
	smsc get maxduedelay

DESCRIPTION
	This command is used to get value for max-due-delay (in seconds). This parameter 
	specifies the maximum possible delay time period in seconds between delivery
	attempts.

EXAMPLES
	smsc get maxduedelay
