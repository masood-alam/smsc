Name
	smsc get removinglivetablesdays

SYNOPSIS
	smsc get removinglivetablesdays

DESCRIPTION
	This command is used to retrieve the current settings for auto-dropping LIVE 
	tables from the Cassandra Database. 

SEE ALSO
	smsc set removinglivetablesdays, smsc set removingarchivetablesdays
	smsc get removingarchivetablesdays
