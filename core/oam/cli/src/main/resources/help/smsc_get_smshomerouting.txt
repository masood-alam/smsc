Name
	smsc get smshomerouting

SYNOPSIS
	smsc get smshomerouting

DESCRIPTION
	The SMSC can be configured to support Home Routing functionality, by setting
	the parameter 'smshomerouting' to true. This command is used to retrieve the
	current settings.
