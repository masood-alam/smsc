Name
	smsc get subscriberbusyduedelay

SYNOPSIS
	smsc get subscriberbusyduedelay

DESCRIPTION
	This command is used to get value for subscriber-busy-due-delay (in seconds).
	This parameter specifies the delay time period in seconds when there has
	been a delivery failure with the cause 'subscriber busy'.

EXAMPLES
	smsc get subscriberbusyduedelay
