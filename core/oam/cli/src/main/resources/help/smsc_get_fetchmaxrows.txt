Name
	smsc get fetchmaxrows

SYNOPSIS
	smsc get fetchmaxrows

DESCRIPTION
	This command is used to get the maximum message fetching count for every fetching
	step from the database.
	The default value is 100 messages.

EXAMPLES
	smsc get fetchmaxrows
