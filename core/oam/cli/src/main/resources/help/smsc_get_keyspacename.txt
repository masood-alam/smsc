Name
	smsc get keyspacename

SYNOPSIS
	smsc get keyspacename

DESCRIPTION
	This command is used to get the Keyspace name for Cassandra Database. If you use
	the script available in the distributive the name will be set to 'RestCommSMSC' by
	default. 

EXAMPLES
	smsc get keyspacename
