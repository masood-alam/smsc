Name
	smsc get modefaultmessagingmode

SYNOPSIS
	smsc get modefaultmessagingmode

DESCRIPTION
	This command is used to get a value for default messaging mode
	value for mobile originated (MO) messages.
	Possible values: 0-default SMSC mode, 1-datagram, 2-transactional,
	3-storeAndForward.

EXAMPLES
	smsc get modefaultmessagingmode

SEE ALSO
	smsc set modefaultmessagingmode, smsc set httpdefaultmessagingmode,
	smsc set hrdefaultmessagingmode, smsc set sipdefaultmessagingmode,
