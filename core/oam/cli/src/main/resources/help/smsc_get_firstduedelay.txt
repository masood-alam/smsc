Name
	smsc get firstduedelay

SYNOPSIS
	smsc get firstduedelay

DESCRIPTION
	This command is used to get value for first-due-delay (in seconds). This 
	parameter specifies the delay time period in seconds between message incoming time 
	and first delivery attempt.

EXAMPLES
	smsc get firstduedelay
